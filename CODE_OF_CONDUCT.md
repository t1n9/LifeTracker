# 行为准则

## 我们的承诺

为了营造一个开放友好的环境，我们作为贡献者和维护者，承诺参与我们的项目和社区的每个人都能够获得一个无骚扰的体验，不受年龄、体形、身体残疾、种族、性别认同和表达、经验水平、国籍、外表、种族、宗教或性身份和定向的骚扰。

## 我们的标准

有助于营造积极环境的行为包括：

- 使用热情友好的语言
- 尊重不同的观点和经验
- 友善接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同情

不可接受的行为包括：

- 性暗示或性骚扰
- 人身攻击、辱骂或贬低
- 骚扰性评论
- 未经同意的私人信息的发布
- 任何可能被合理认为不适当的其他行为

## 执行

项目维护者有责任阐明可接受行为的标准，并应对任何不可接受行为采取适当的纠正措施。

项目维护者有权删除、编辑或拒绝与本行为准则不一致的评论、提交、代码、wiki 编辑、问题和其他贡献，或暂时或永久禁止任何贡献者的其他行为。

## 范围

本行为准则适用于项目空间和公开代表项目的空间。代表项目的例子包括使用官方项目电子邮件、通过官方社交媒体账户发布，或在线上或线下事件中担任指定代表。

## 报告

遇到不可接受行为的实例可以通过 GitHub Issues 私下报告给项目维护者。所有投诉都将得到审查和调查，并将得到认为必要和适当的回应。
